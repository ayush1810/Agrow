#### Installation
---
> Clone the repo  
    ```
    git clone https://github.com/ayush1810/agrow.git
    ```

> Setup Mongo uri in a variable.env, then install dependencies
    ```
    npm install 
    ```
> Compile
    ```
    npm run compile 
    ```
> Run     
    ```
    npm start
    ```
